# Smart Complaint System

## Overview
This is a backend system for Smart Complaint System built using Flask and MySQL.  
It handles complaint submission, storage, and management.

## Tech Stack
- Python (Flask)
- MySQL
- HTML, CSS, JavaScript
## Database
Database schema is provided in database.sql

## Features
- User login system
- Complaint submission
- Data stored in MySQL database
- Backend API handling

## Project Structure

- app.py → Main Flask application  
- templates/ → HTML files  
- static/ → CSS, JS, images  
- database.sql → Database schema
